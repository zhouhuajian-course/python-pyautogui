Python PyAutoGUI UI自动化
==================

课程介绍
-------

PyAutoGUI 是Python比较强大的UI自动化库，有鼠标控制、键盘操作、屏幕截图、图片定位、消息对话框、窗口操作等功能。  

库里面还有很多非常实用的工具，例如倒计时、获取坐标信息、迷你语言执行UI自动化等。这个库还有故障安全机制、通用暂停机制等。  

本课程，用讲解PyAutuGUI源码以及官方文档的方式，介绍库里面的功能函数，并带大家一起开发一个UI自动化项目。
