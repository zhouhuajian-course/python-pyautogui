PyCharm 快捷键
-----------------

1. Ctrl + Shift + F10  运行
2. Ctrl + D  复制选定的区域或行
3. Ctrl + /   行注释/取消行注释
4. Alt + 1  项目
5. Alt + 7  代码结构
6. Alt + F12  终端
